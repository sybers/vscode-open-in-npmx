# Open in npmx.dev

<p align="center">
  <img src="https://raw.githubusercontent.com/sybers/vscode-open-in-npmx/master/resources/logo.png" width="128" alt="Logo">
</p>

Open the current selection, project, or arbitrary string, in [npmx.dev](https://npmx.dev)

## Install

Follow the instructions in the [Marketplace](https://marketplace.visualstudio.com/items?itemName=sybers.vscode-open-in-npmx), or run the following in the command palette:

```shell
ext install sybers.vscode-open-in-npmx
```

## Usage

This extension adds the following command to the commands palette:

```js
"Open in npmx.dev"; // Open the current selection, project, or arbitrary string, in npmx.dev
```
